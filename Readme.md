# Mouse Controller

暂不支持侧键。  
键控软件，使用方法如下，

## 准备环境(python 3.11.7)

```
$ git clone https:
$ pip install -r requirements.txt
```
## 运行
```
$ python3 main.py
```

## 使用说明

- `left_alt`开始录制  
- `F2`停止录制  
- `left_ctrl`开始执行
- `left_shift`中断
- `Tab`读取所有历史数据
